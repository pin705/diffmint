# Provider Strategy

Each workspace should choose one of two provider modes:

- Managed provider by the Diffmint platform
- Bring your own key with explicit model allow-lists

Recommended defaults:

- one primary provider
- one fallback provider
- default model pinned per workspace
- rate limit by workspace
- secret rotation and audit ownership defined up front
