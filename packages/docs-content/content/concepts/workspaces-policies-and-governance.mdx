# Workspaces, Policies, and Governance

Diffmint is not positioned as a general AI assistant. It is a governed review system for teams.

## Workspace scope

Each workspace owns:

- provider strategy
- plan and quota
- team rules
- synced review history
- audit events

## Policy scope

Each review can carry:

- policy version ID
- workspace metadata
- provider and model metadata
- trace ID

That bundle makes review output searchable, enforceable, and auditable later.
