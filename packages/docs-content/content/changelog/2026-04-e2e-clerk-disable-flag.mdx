Diffmint now supports `DIFFMINT_DISABLE_CLERK` and `NEXT_PUBLIC_DIFFMINT_DISABLE_CLERK` for local
production smoke tests. This keeps Playwright and manual `next start` verification deterministic
even on machines that already have Clerk development keys in `apps/web/.env`.
