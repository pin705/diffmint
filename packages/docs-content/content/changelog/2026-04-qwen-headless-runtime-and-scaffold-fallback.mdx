# Qwen headless runtime and scaffold fallback

- `dm review` now routes through the Qwen headless runtime when a local `qwen` binary is available.
- If Qwen is missing, unauthenticated, or explicitly disabled with `DIFFMINT_REVIEW_RUNTIME=scaffold`,
  Diffmint falls back to the built-in scaffold reviewer.
- Unit tests now pin CLI integration tests to scaffold mode and cover the mocked Qwen runtime path
  to keep the test suite deterministic.
