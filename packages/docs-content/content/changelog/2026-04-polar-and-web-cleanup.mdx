# 2026-04 Polar Billing and Web Cleanup

This update tightens the Devflow web app around the actual product direction.

- Polar now backs billing configuration, checkout, customer portal, and webhook routes
- the billing page is a Devflow control-plane surface instead of a starter demo pricing table
- legacy demo routes for products, users, kanban, chat, notifications, forms, and react-query were removed
- the dashboard overview route now renders a single Devflow control-plane page instead of starter parallel charts
- auth, privacy, and terms copy now reflect Devflow rather than the original dashboard starter
