# Strict Persistence Mode

- added `DIFFMINT_REQUIRE_PERSISTENCE=true` for production environments that must not fall back to in-memory control-plane state
- service-layer persistence now fails fast when Postgres is required but unavailable
- readiness checks now report a failing database check when strict persistence is enabled without an active database
