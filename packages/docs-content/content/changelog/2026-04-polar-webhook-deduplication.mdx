# Polar webhook deduplication

- Repeated Polar webhook deliveries are now deduplicated in the control-plane service.
- Duplicate deliveries no longer create extra audit entries or re-apply the same billing seat update.
- The billing guide now documents this retry-safe behavior for production webhook operations.
