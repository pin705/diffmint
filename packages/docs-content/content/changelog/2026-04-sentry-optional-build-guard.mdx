# Sentry optional build guard

- Diffmint now keeps Sentry off by default unless the optional runtime and build credentials are configured.
- The web build only enables the Sentry plugin when the DSN, org, project, and auth token are all present.
- This keeps local and CI builds closer to the documented optional-integration behavior.
