# Cloud Sync Redaction Hardening

- synced review history now redacts common bearer tokens, API keys, GitHub tokens, and secret-style assignments before upload
- raw provider output artifacts are replaced with a redacted placeholder before they reach cloud history
- control plane ingest applies the same sanitization path as a defense-in-depth safeguard
