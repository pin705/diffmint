# Brand, Landing, and SEO Refresh

- the public web surface now uses a dedicated Diffmint mark instead of a generic command icon
- the landing page has been rebuilt as a terminal-first marketing page with clearer product positioning and install calls to action
- Next.js file-based metadata now covers icons, Apple icon, Open Graph image, Twitter image, robots, sitemap, and web manifest
- the VS Code extension icon now matches the refreshed Diffmint brand mark
