# Sentry Build Noise Reduction

- builds now alias `@sentry/nextjs` to a no-op runtime when Sentry monitoring is disabled
- webpack now ignores a narrow set of known upstream OpenTelemetry warning patterns from the Sentry stack
- local production smoke runs are quieter without changing the live Sentry path when monitoring is enabled
