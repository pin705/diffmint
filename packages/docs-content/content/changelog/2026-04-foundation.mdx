# 2026-04 Foundation Release

This release establishes the production foundation for Diffmint:

- Bun monorepo with `apps/web`, `apps/cli`, `apps/vscode`, and shared packages
- public docs and dashboard docs center
- shared contracts for review, policy, history, and release metadata
- Drizzle schema and control-plane API skeleton
- CLI and VS Code scaffolds aligned to the same runtime model
