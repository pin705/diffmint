# Production build hardening

- Web production builds now run through Next.js webpack mode for more predictable CI and release builds.
- Local development keeps the regular `next dev` flow, so hot reload behavior stays unchanged.
- This change pairs with the optional Sentry build guard to keep production build prerequisites explicit.
