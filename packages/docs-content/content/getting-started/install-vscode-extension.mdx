# Install the VS Code Extension

The extension is a companion to the CLI. It shells out to the local `dm` binary and renders results in editor-native panels.

## Setup flow

1. Install the CLI first.
2. Install the VS Code extension.
3. Confirm the extension can find the `dm` binary, or set `diffmint.cliPath`.
4. Set `diffmint.webBaseUrl` if you are pointing the extension at a self-hosted or local control plane instead of `https://diffmint.io`.
5. Click **Sign In** in the extension.
6. Complete browser auth and return to the editor.
7. Run **Review Current Changes**.

## Expected commands

The extension uses the same flows as the CLI:

- `dm review`
- `dm review --staged`
- `dm review --files ...`
- `dm explain <file>`
- `dm tests <file>`

Read the [VS Code guide](/docs/vscode/guide) for command mapping and result panels.

## Packaging

For internal rollout or release automation, package the extension as a VSIX:

```bash
pnpm package:vscode
```
