# 5-Minute Quickstart

Diffmint is a local-first review product. The primary experience lives in the CLI and VS Code extension. The web app exists to manage workspace settings, policies, billing, history, and audit.

## Developer quickstart

1. Install the CLI.
2. Run `dm auth login`.
3. Sign in in the browser, select the right workspace if prompted, then approve the device session.
4. Run a local review against your current branch.
5. Open synced history in the dashboard.

```bash
pnpm install
dm auth login
dm review --base origin/main
dm history
dm doctor
```

If the control plane is temporarily unavailable, Diffmint keeps review sync items locally and flushes
them automatically on the next successful workspace-connected command.

## Admin quickstart

1. Create a workspace in the control plane.
2. Choose a provider strategy.
3. Publish a policy version.
4. Invite the team.
5. Share the CLI and VS Code install docs.

Read [Install the CLI](/docs/getting-started/install-cli) and [Install the VS Code extension](/docs/getting-started/install-vscode-extension) before rollout.

For a one-command local stack with Postgres and hot reload, use [Docker development](/docs/getting-started/docker-development).
